In our seminal paper modelling Prospective Memory, we discussed how it may be possible to use our model to identify lapses of memory. Since then, I have implemented and tested the code to do that. The code is implemented in the Dynamic Models of Choice (DMC) suite of functions.

Press here to download the model code. As with all custom DMC models, the user must add it to the 'models' directory. It is a variant of the Linear Ballistic Accumulator (LBA) model, and so it makes sense to store it in the LBA subdirectory. As with all DMC model files, the code defines a transform, likelihood, and random function. Below I go through the code piece by piece.
